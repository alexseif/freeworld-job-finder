# Contributing to Freeworld Job Finder

Thank you for your interest in contributing to Freeworld Job Finder! We are building an open-source, ethical, AI-driven job scraper and pipeline.

## Code of Conduct
By participating in this project, you are expected to uphold graceful and professional open-source standards. Please be respectful, constructive, and inclusive in all communications and pull requests.

## How to Contribute
1. **Fork & Branch:** Fork the repository and create a new branch for your feature or bug fix.
2. **Adhere to Standards:** Ensure your code strictly adheres to our PSR-12, Symfony, and Composer guidelines outlined in `guidingprinciples/STANDARDS.md`.
3. **Commit Messages:** Write clear, concise commit messages detailing the "why" and "what" of your changes.
4. **Pull Requests:** Submit your PR with a comprehensive description. Reference any related issues.

## Reporting Bugs
Please use the GitHub Issue Tracker. Provide detailed reproduction steps, logs, and your environment context.
