# AI Context: php-jobspy

## Project Overview
`php-jobspy` is a standalone, decoupled PHP library built to achieve feature parity with the `python-jobspy` project. It scrapes job boards (LinkedIn, Indeed, Glassdoor) without relying on official APIs.

## Current State & Roadmap
- We use `guzzlehttp/guzzle` for all HTTP requests.
- **LinkedInScraper** is actively implemented parsing the public SERP via DOMXPath.
- The roadmap and missing features are strictly tracked in `PYTHON_JOBSPY_PARITY.md`. Always check this file before implementing new features.

## Architectural Rules
- **No Native cURL:** Use Guzzle for all network requests. Enforce strict HTTP timeouts.
- **Graceful Failure:** Scrapers must catch 429/999 AuthWall HTTP errors and return an empty array rather than throwing fatal exceptions that crash the parent pipeline.
- **Data Normalization:** All returned job arrays must map precisely to the standard schema: `title`, `company`, `location`, `is_remote`, `date_posted`, `time_ago`, `is_easy_apply`, `url`, `description`.
